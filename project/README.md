## 1. Паспорт проекта

- **Название проекта:** Классификация тональности отзывов (Sentiment Analysis)
- **Автор:** Степанов Матвей Евгеньевич
- **Группа:** БАСО-02-24
- **Контакт:** s9175954703@yandex.ru

- **Краткое описание (2-4 предложения):**  
  Проект решает задачу определения тональности текстового отзыва (положительный / отрицательный).  
  Используется открытый датасет отзывов IMDB, модель логистической регрессии на признаках TF-IDF.  
  Результат – REST API сервис на FastAPI, который принимает текст и возвращает тональность и уверенность.

---

## 2. Структура проекта

Проект организован в следующей структуре:

- `requirements.txt` – зависимости проекта.
- `report.md` – отчёт по проекту.
- `self-checklist.md` – чеклист самопроверки.
- `notebooks/` – ноутбук с EDA и обучением модели (`exp01_eda_and_model.ipynb`).
- `src/` – основной код: `model.py` (загрузка модели и предсказание), `service.py` (API-сервис).
- `data/raw/` – папка с данными (файл `IMDB_Dataset.csv` не коммитится, только инструкция по скачиванию).
- `configs/` – файл `.env.example` (шаблон переменных окружения).
- `tests/` – тесты (`test_service.py`).
- `artifacts/` – сохранённые модель и векторизатор (`model.pkl`, `vectorizer.pkl`) – не коммитятся.

Структура соответствует базовой.

## 3. Требования и установка

### 3.1. Требования

- Python `>= 3.10`
- Установленный `git` (для клонирования репозитория)

### 3.2. Установка окружения

```bash
# В терминале перейти в папку проекта
cd project

# Создать виртуальное окружение
python -m venv .venv

# Активировать окружение:
# Windows:
.venv\Scripts\activate
# Linux / macOS:
source .venv/bin/activate

# Установить зависимости
pip install --upgrade pip
pip install -r requirements.txt

# Обучить модель (если артефактов нет)
python src/train.py

# Запустить сервис
python -m src.service
